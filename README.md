# CANUT Social Planner

Gestionnaire de publications Facebook pour CANUT PC.

## Fonctionnalités

- 📅 Calendrier éditorial
- 📝 Gestion des publications
- 💡 Banque d'idées
- 📊 Statistiques
- 🔍 Recherche de sujets
- 📤 Export JSON

Développé en HTML, CSS et JavaScript.

Aucune base de données.
Compatible GitHub Pages.
